# Trie Algorithms and Data Structures II - UFABC

## Requisitos
* NodeJS 18.x

Para a execução dos comandos abaixo você deve estar na pasta do projeto, ou seja,
na pasta que contém o arquivo: __package.json__

## Instalação dos pacotes
```bash
npm install
```

## Build e Execução

```bash
  npm run build
  node ./dist/main.js
```
